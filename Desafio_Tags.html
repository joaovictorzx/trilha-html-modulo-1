O que é o 'HTML'

Voltar

HTML é a sigla para HyperText Markup Language, que significa 'Linguagem de Marcação de Hipertexto'. É a linguagem padrão para criar páginas web. O HTML descreve a estrutura de uma página usando marcações, que são elementos que definem diferentes partes do conteúdo, como títulos, parágrafos, links, imagens e outros elementos multimídia.

"O HTML é uma linguagem de marcação que estrutura a web, permitindo que o conteúdo seja organizado e compreendido pelos navegadores e usuários." - Tim Berners-Lee1

História do HTML

Voltar

O HTML foi criado por Tim Berners-Lee em 1991 como parte do desenvolvimento da World Wide Web. Desde então, passou por várias versões e atualizações, com o objetivo de melhorar a funcionalidade e a acessibilidade das páginas web. A versão mais recente é o HTML5, que introduziu novas funcionalidades, como suporte a áudio e vídeo, gráficos vetoriais e APIs para aplicativos web.2

Estrutura do HTML

Voltar

A estrutura básica de um documento HTML inclui as seguintes partes:

  1. <!DOCTYPE html>: Declara o tipo de documento.
  2. <html>: Elemento raiz que envolve todo o conteúdo da página.
  3. <head>: Contém metadados, título da página e links para recursos externos.
  4. <body>: Onde fica o conteúdo visível da página, como textos, imagens e links.

Tags semânticas do HTML5

Voltar

As tags semânticas do HTML5 são elementos que descrevem o significado do conteúdo, melhorando a acessibilidade e a SEO (Search Engine Optimization). Algumas das principais tags semânticas incluem:

  1. <header>: Define o cabeçalho da página ou de uma seção.
  2. <nav>: Indica uma área de navegação com links.
  3. <main>: Representa o conteúdo principal do documento.
  4. <section>: Agrupa conteúdos relacionados em seções.
  5. <article>: Utilizado para conteúdos independentes, como artigos ou posts.
  6. <aside>: Conteúdo complementar, como barras laterais.
  7. <footer>: Define o rodapé da página ou de uma seção.

Tipos de textos HTML

Voltar

O HTML permite a formatação de textos de várias maneiras, incluindo:

  1. <h1> a <h6>: Títulos e subtítulos, com <h1> sendo o mais importante.
  2. <p>: Parágrafos de texto.
  3. <strong>: Texto em negrito.
  4. <i>: Texto em itálico.
  5. <mark>: Texto destacado com um marcador.
  6. <blockquote>: Citações longas, geralmente em itálico e centralizadas.
